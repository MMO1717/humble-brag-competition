# 回复策略指南 / Response Strategy Guide

## English Prompt Notes

Default to light_acknowledgment when no stronger social constraint applies.

Use neutral_observation for workplace_channel, academic_forum, public_social_media, stay_neutral, or professional goals.

Use humor_tease only for close_friend plus private_chat or group_chat when the brag is clearly self-aware or playful.

Use ask_followup when a cautious direct-message reply should avoid overpraise and a grounded question fits the context.

Do not actively choose set_boundary or no_response in the baseline unless the situation clearly requires it.

The response_text must match the chosen response_strategy.
