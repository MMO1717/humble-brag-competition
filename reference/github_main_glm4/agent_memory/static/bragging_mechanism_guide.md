# 炫耀机制指南 / Bragging Mechanism Guide

## English Prompt Notes

humble_complaint: complaint-like wording that indirectly highlights an advantage, burden, status, attention, or difficulty.

faux_modesty: explicit modesty, self-deprecation, or a disclaimer that draws attention to success. Use only when the modesty cue is obvious.

achievement_drop: casual mention of an achievement, status, opportunity, score, award, role, or named accomplishment.

comparison_superiority: implying the speaker is better, faster, smarter, more skilled, or more efficient than others.

scarcity_flex: highlighting rare access, exclusive opportunity, limited availability, or privileged access.

understated_flex: low-key framing of a positive outcome or ability without explicit modesty. Prefer this over faux_modesty when no clear modesty cue exists.

self_aware_brag: openly joking about bragging or flexing, such as "not to brag", "don't mean to brag", "humblebrag", or "flex".

other: edge cases that do not fit the above.

Tie-breakers: do not overuse faux_modesty. If the post says "not to brag", choose self_aware_brag. If a complaint functions as proof of status or ability, choose humble_complaint.
